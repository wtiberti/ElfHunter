# ![logo](imgs/github_banner.png) ElfHunter

## Info

An open-source ELF files analyzer/editor.

## Requirements

+ Platform: Linux/Unix
+ GCC >= 4.8.0 ( libm.so.6, libgcc\_s.so.1, libc.so.6 )
+ QT >= 4.8.4 ( libQtGui.so.4, libQtCore.so.4 )
+ libokteta1core.so.1
+ liboktetagui.so.1
